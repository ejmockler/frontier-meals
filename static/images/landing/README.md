# Landing Page Images

This directory contains images for the Frontier Meals landing page.

## Required Images

The following images need to be placed in this directory:

### Hero Section
- `hero-community-lunch.jpg` (721x1280 portrait) - Main hero image showing community lunch at Frontier Tower

### Community Gallery
- `rooftop-celebration.jpg` (1280x721 landscape) - Rooftop lunch celebration scene
- `happy-participants.jpg` (1280x721 landscape) - Happy lunch participants
- `community-dining.jpg` (721x1280 portrait) - People dining together

## Image Guidelines

- Use authentic, warm photos that convey community and belonging
- Ensure images are optimized for web (compressed but high quality)
- Hero image should be loaded eagerly (above the fold)
- Gallery images use lazy loading (below the fold)
